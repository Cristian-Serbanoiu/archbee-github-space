# INDENTED OUTLINE PATTERNS

An outline is easiest to read when indentation is consistent. This page focuses on nested bullets and nested list markers inside numbered items.

## Outline template (single theme)

- Section
  - Purpose statement
  - Audience statement
  - Included list examples
    - Bulleted list
      - Level 3 example
    - Numbered list
      - Level 3 example
    - Checklist list
      - Level 3 example

## Example: release notes structure

1. Summary
   - One-sentence change overview
2. What changed
   - Added
   - Fixed
   - Improved
3. How to verify

- [ ] Run smoke test
- [ ] Validate edge cases
- [ ] Confirm copy/paste friendly formatting

## Indentation stress test

- Formatting edge cases

* [ ] Bullet inside checklist item
* [ ] Checklist inside a numbered context
